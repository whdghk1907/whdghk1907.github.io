# 조건문  
<br>

## if else
<br>

```javascript
const a = `a`;
if (a === `b`){
  console.log(`a는 b입니다.`);
} else if (a === `c`){
  console.log(`a는 c입니다.`);
} else {
  console.log(`모르겠습니다.`);
}

console.log(a === `b` ? `a는 b입니다.` : `a는 b가 아닙니다.`);
```  
<br>
if 는 ()안에 조건문을 적고,  
이 값이 true이면 {}의 내용을 실행한다.  
<br>
간단한 조건문의 경우에는, 예시처럼
 ?와 :를 이용하여도 가능하다.

## switch
<br>

```javascript
const a = `a`;

switch(a) {
  case `a`:
    console.log(`a 입니다.`);
    break;
  case `b`:
  case `c`:
    console.log(`a 가 아닙니다.`);
    break;
  default:
    console.log(`a를 확인하세요`);
}

```  
<br>
switch 선언 후 ()안에 변수를 넣고  
case를 통하여 값을 검사한다.  
값이 일치하여 true가 나올 경우,  
case에 할당된 : 의 내용이 실행된다.  
default를 통해 기본값을 정해줄 수 있다.  
<br>
여러가지 조건을 한번에 검사할 때  
유용하다.
